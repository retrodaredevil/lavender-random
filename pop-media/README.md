# Pop Media

These scripts aim to help setup, or keep up to date a Pop!\_OS 
install that focusses on media play back.

Apps that are setup include:

* PlexAmp
* Plex HTPC

## Manual Setup

### PlexAmp

Go to PlexAmp settings

* Device >
  * Player Name: Miles
  * Remote Control >
    * Remote Control: Yes
* Experience >
    * Grid Size: Large
    * Font Size: Mega
